# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 2.x     | ✅        |
| 1.x     | ❌        |

## Reporting a Vulnerability

**Do not open a public issue for security vulnerabilities.**

Email: nadimtuhin@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

You'll receive a response within 48 hours. Fixes are prioritized and coordinated before public disclosure.
