# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2024

### Added
- PDF accessibility report export via Puppeteer
- Extended a11y checks: ARIA roles, focus trap, target size, lang attr, skip nav, color blindness, labels
- WCAG level filtering (A, AA, AAA)
- CLI support

## [1.0.0] - Initial release

### Added
- Basic accessibility crawling with axe-core
- Puppeteer integration
